# 🦐 Shrimp Sweep: Ocean Acidification Game

An interactive browser-based game that teaches ocean acidification concepts through gameplay. Players guide a shrimp through increasingly difficult reef mazes while answering marine biology questions to unlock deeper levels.

---

## 🌊 Overview

Shrimp Sweep is a PyScript-powered game built for a marine biology course. It combines:

- 🧠 Quiz-based progression  
- 🎮 Pac-Man–style movement  
- 🌡️ Ocean acidification concepts  
- 📊 Leaderboards and performance tracking  

Players must answer **3 questions correctly in a row** to advance through levels representing worsening ocean conditions.

---

## 🎯 Gameplay Features

- 🦐 Control a shrimp through a reef maze  
- 🦑 Avoid squid enemies (which get faster each level)  
- 💣 Collect carbon dioxide (temporarily flips gameplay)  
- 🪱 Collect worms (slow enemies, boost shrimp)  
- • Collect plankton for points  
- 🧠 Answer quiz questions to unlock levels  

---

## 🧪 Levels (Ocean Acidification Theme)

- **Easy — pHresh & CO₂**  
  Stable ocean conditions, introductory questions  

- **Medium — CO₂ncerning pH**  
  Increasing acidification, more complex concepts  

- **Hard — pH-anic Mode**  
  Severe ocean stress, advanced biology questions  

---

## 🏆 Leaderboards

- 🎣 **Catch of the Day** — Best overall run  
- 🧪 **Acidification Leaders** — Fastest full completion  
- 🦐 **Shrimp Showdown** — Highest quiz accuracy  
- 🔬 **Calcification Rankings** — Longest survival
